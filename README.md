# TRS-Monitor
Manufacturing Execution System (MES) – Suivi et pilotage du TRS

Ce projet consiste en le développement d’un MES (Manufacturing Execution System) léger, permettant de collecter, exploiter et analyser les données de production afin de piloter la performance industrielle.

L’outil permet de :

Renseigner les cadences de production en temps réel ou différé
Calculer automatiquement le TRS (Taux de Rendement Synthétique) des lignes de production
Importer et lire des rapports de production existants pour en extraire les données clés
Intégrer ces données historiques afin de recalculer les TRS et enrichir l’analyse globale
Fournir une vision data-driven du chantier, facilitant la prise de décision et l’amélioration continue

L’objectif principal est de transformer des données de production brutes en indicateurs de performance fiables et exploitables, afin d’optimiser le pilotage des lignes industrielles et d’identifier rapidement les leviers de progrès.

Manufacturing Execution System (MES) – OEE Monitoring & Performance Management

This project involves the development of a lightweight Manufacturing Execution System (MES) designed to collect, process, and analyze production data in order to improve industrial performance management.

The tool enables:

Recording production rates in real-time or retrospectively
Automatically calculating OEE (Overall Equipment Effectiveness) for production lines
Importing and reading existing production reports to extract key data
Integrating historical data to recalculate OEE and enhance overall analysis
Providing a data-driven approach to operations, supporting decision-making and continuous improvement


The main objective is to transform raw production data into reliable and actionable performance indicators, helping optimize production line management and quickly identify improvement opportunities.
